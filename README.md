# LHR_REBOOT

**Left Hand Right** - Unreal Engine 5.6 Project

## Project Information

- **Engine Version:** Unreal Engine 5.6
- **Project Name:** LHR_REBOOT
- **Last Updated:** See DEVLOG.md for version history

## Project Overview

A game project featuring:
- Good/Bad tree mechanics (BP_GoodTree, BP_BadTree)
- Seed projectile system (BP_GoodSeedProjecticle, BP_BadSeed_Projectile)
- Animal AI system (BP_Animal, Bad_Character_AI)
- Growth stone mechanics (BP_GrowthStone)
- Character system (LHR_CharacterBP)
- HUD elements (Main Menu, Stats, Timer)
- Multiple maps and environments

## Assets

The project includes various asset packs:
- Animal Variety Pack (Crow, Deer, Fox, Pig, Wolf)
- Arrow Trail VFX
- Basic VFX
- Black Alder foliage
- Field Grass Pack
- Nanite Environment Bundle
- Interactive Spruce Forest

## Maps

- `LHR_DevMap` - Development/testing map
- `LHR_SimpleMap` - Simple map (set as default)
- `L_LHR_Ring3_FogTrees` - Ring map with fog and trees

## Setup

1. Open `LHR_REBOOT.uproject` with Unreal Engine 5.6
2. The project will compile C++ code if needed
3. Default startup map: `LHR_SimpleMap`

## Plugins

- **ModelingToolsEditorMode** - Enabled for Editor
- **GameplayStateTree** - Enabled
